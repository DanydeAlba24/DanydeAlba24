# Who am I?

Mi perfil combina una sólida base técnica en infraestructura de sistemas, capacidades técnicas para análisis de datos y desarrollo de base de datos, además de un perfil híbrido fullstack sumado a la habilidad de 
comunicar y capacitar a usuarios técnicos y no técnicos.

agilidad de aprendizaje y capacidad de comunicación.
