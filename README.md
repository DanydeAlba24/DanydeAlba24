# Who am I?

**Soy Ingeniero en Sistemas Computacionales con una especialización en curso sobre Implementación de Servicios de Oracle.**

Mi perfil combina una sólida base técnica en infraestructura de sistemas, capacidades técnicas para análisis de datos y desarrollo de base de datos, además de un perfil híbrido fullstack sumado a la habilidad de 
comunicar y capacitar a usuarios técnicos y no técnicos.

Mi personalidad me permite mantenerme a la vanguardia con proyectos prácticos en AWS, Node.js y arquitecturas SaaS asimismo como optimización de recursos y una combinación de versatilidad técnica, 
agilidad de aprendizaje y capacidad de comunicación.
